### README

​		本次模型使用`3_train_regression.csv`文件作为训练集，`regression_test.csv`作为测试集，train文件夹中包含了其他的数据集，也可以将它们当作测试集或训练集用。

​		`regression.ipynb`和`regression.py`代码内容相同，是模型的训练代码；

​		`test_interface_regression.ipynb`是测试接口，用来测试模型误差。
